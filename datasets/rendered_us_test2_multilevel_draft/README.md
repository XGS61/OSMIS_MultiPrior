# Review annotation for the included test2 case

This folder contains the hierarchical annotation draft used by the current
Full-SEAN experiment.

- `review_panel.png`: source, overlay, and color-label review sheet.
- `multilevel_mask_indexed.png`: six exclusive display/style regions.
- `hierarchical_conditions.npz`: overlapping structural channels consumed by
  `prepare_hierarchical_case.py`.
- `channel_*.png`: individual channels for review.

The supplied levator-hiatus annotation is preserved. The three internal
candidate regions were derived from image position and dark-intensity
components and have not been independently clinically validated.
